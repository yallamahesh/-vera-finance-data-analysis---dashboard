# Vera Finance Sales Analysis Dashboard (MS Excel,sql,power bi)

Project Title: Vera Finance Sales Analysis Dashboard

📊 Which branch generates the most profit?

🏍️ Which bike model is most financed?

⚠️ What is the overdue rate across regions?

💰 What is the average loan amount and EMI?

📅 Which month has the highest collections?

📈 What is the relationship between interest rate and EMI status?


# Key Responsibilities:

Cleaned and validated sales and customer data using Excel functions.
Built Pivot Tables and Pivot Charts to summarize sales performance.
Created interactive dashboards using slicers and timelines.
Analyzed branch-wise loan disbursements, collections, and revenue trends.
Monitored EMI payment status (Paid, Pending, Overdue).
Generated monthly and quarterly performance reports for management.
Used XLOOKUP, SUMIFS, COUNTIFS, IF, and other Excel formulas to automate calculations.
Applied conditional formatting to highlight overdue accounts and key KPIs.
Identified top-performing branches and sales executives based on business metrics.
Improved reporting accuracy and reduced manual reporting effort through dashboard automation.
